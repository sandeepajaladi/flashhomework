Introduction to Flask Flash messages

I am Sandeepa Jaladi . This is my first Flask flash message . The purpose of this page is to learn to Flask

Here is the introduction to Flask flash

  1. Flask provides a really simple way to give feedback to a user with the flashing system.
  2. The flashing system basically makes it possible to record a message at the end of a request and access it next request and only next request.
  3. This is usually combined with a layout template that does this.
  4. flashing messages that are too large for session cookies causes message flashing to fail silently.
